Яндекс.Практикум Когорта 1. Александр Демин. Проект 12 спринта v1.0.0
=====================================================================

https://rationabilis.github.io/be12/public/
------------------------------------------------------

Проект инициирован для выполнения задания 12 спринта Яндекс.Практикума в сентябре 2019 г., как Back End продолжение разработки предыдущих спроинтов.

В основе использован express. В проекте есть:

- package.json;
- .editorconfig;
- .eslintrc, расширяющий конфигурацию airbnb-base, а также необходимые для работы линтера dev-зависимости;
- .gitignore;
- команда npm run start запускает сервер на localhost:3000;
- команда npm run dev запускает сервер на localhost:3000 с хот релоудом;
- адрес localhost:3000 загружает фронтенд проекта Mesto, который вы делали;
- в ответ на запрос GET localhost:3000/users сервер вернёт JSON-объект из файла users.json;
- в ответ на запрос GET localhost:3000/cards сервер вернёт JSON-объект из файла cards.json;
- в ответ на запрос GET localhost:3000/users/8340d0ec33270a25f2413b69, сервер вернёт JSON-объект пользователя с переданным после /users идентификатором;
- если пользователя с запрошенным идентификатором нет, API должен возвращать 404 статус ответа и JSON: { "message": "Нет пользователя с таким id" };
- при запросе на несуществующий адрес, API должен возвращать 404 статус ответа и JSON: { "message": "Запрашиваемый ресурс не найден" }.
